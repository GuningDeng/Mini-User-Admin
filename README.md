# Mini User Admin  
Een eenvoudig mini-gebruikersbeheer systeem ontwikkeld door PHP. Front framework: Bootstrap. Database: MySQL.   

**Realisatie:**  
+ CRUD  
+ Log in  
+ Register  
+ Welkomstpagina  
+ Gebruikers lijst  
+ Voeg gebruiker toe  
+ Gebruikersinformatie bijwerken  
+ Verwijder gebruiker  
+ Routes & sessiebeheer   
+ Input verificatie  
+ Anti-SQL-injectie
